# 🚀 vicky.dev

> Personal portfolio — projects, blogs, and everything I build.

## 🛠️ Tech Stack

- **React.js** — UI framework
- **Tailwind CSS** — Styling
- **Framer Motion** — Animations
- **React Router** — Navigation
- **Vite** — Build tool

## 📄 Sections

- **Hero** — Intro, animated role switcher, CTA buttons
- **About** — Bio, skills, what I'm currently building
- **Projects** — Showcase with tech stack, live demo & GitHub links
- **Blog** — Articles on what I learn and build
- **Experience** — Internships, freelance, and milestones timeline
- **Stack** — All tools and technologies I use
- **Contact** — Email, LinkedIn, GitHub

## 🚧 Getting Started

```bash
git clone https://github.com/vicky/vicky.dev.git
cd vicky.dev
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
```

## 🌐 Deployment

Deployed on **Vercel** — auto deploys on push to `main`.
